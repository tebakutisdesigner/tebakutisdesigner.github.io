Lua: Choplifter - Zombie Apocalypse

During the semester I spent learning Lua programming, our final project was to take all of the individual functions we had learned to create (saving and loading games, a player shop, and so on) and use those along with new functions to create a “total conversion” of our instructor’s hack-and-slash game, LuaQuest. After my instructor approved my GDD, I coded all changes in Lua to complete my game. You may download a complete set of the LuaScripts I updated and created for the game. Each is viewable in Notepad.

Lua: Table Search (Parser)

As an in-class assignment, I coded a parser that would take an input string and a table from the user, search for that input string within that table, and either return the index at which it found the input string, or tell the user the input string does not exist in the table. Lua: Combat Round with Variable Items and Stats

As one of the many assignments for my Lua programming class, I created a combat function to handle a round of combat between an attacker and defender. I had to properly incorporate unit stats (such as strength and defense), determine what weapon each unit was using, determine what armor each unit was wearing, and then complete a round of combat that incorporated these values. I ended each round by subtracting HP from each unit (assuming both units were still alive) and killing the defending unit if it hits 0 HP or below.
